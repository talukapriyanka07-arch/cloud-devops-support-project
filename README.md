# Cloud Operations & DevOps Support Project

## Overview
This project demonstrates foundational cloud operations and DevOps practices.

## Key Activities
- Created CI/CD pipeline using GitHub Actions
- Implemented automated validation for application files
- Simulated cloud monitoring checklist
- Documented IAM access control process
- Created operational runbook for troubleshooting

## Tools Used
- GitHub
- GitHub Actions
- YAML
- Basic HTML

## Learning Outcome
- Understanding of CI/CD workflows
- Version control basics
- Automation fundamentals
- Cloud operational documentation practices
